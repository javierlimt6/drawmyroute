# PathArt

AI-Powered Route Drawing Application.

## Getting Started

### Frontend
The frontend is built with Next.js and located in the `frontend` directory.

To run the development server:

```bash
cd frontend
npm install # if not already installed
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

**Note:** You need a Mapbox Public Access Token in `frontend/.env.local` for the map to render correctly.
